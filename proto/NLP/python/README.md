# Traitement NLP avec Python

## Dépendances 
- Python3.10
- Pip

## Installation
```bash
pip install -r requirements.txt
```

## Utilisation

```bash
python3.10 main.py --text "<Texte à traiter>"
```

À noter que divers fichiers nécessaire au fonctionnement du programme seront téléchargés la première fois
